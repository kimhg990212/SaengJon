## 2. 백엔드 구조 전면 개편

### 설정 모듈 (`config.py`) 단순화
- `pydantic-settings` + `lru_cache` → `python-dotenv` + `os.getenv()` 직접 로드
- `NTS_API_KEY` 누락 시 서버 기동 시점에 `RuntimeError` 즉시 발생
- `CORS_ORIGINS` 파싱에 `.strip()` 추가 → `.env` 공백 오염 방지

### 삭제된 파일
`database.py`, `models/business.py`, `routers/health.py`, `schemas/common.py`, `services/__init__.py`, `utils/__init__.py`, `.env.example`

|------|------|
| `backend/app/db.py` | psycopg2 커넥션 풀 (min 1 / max 10) |

## 3. 국세청(NTS) 사업자등록정보 진위확인 및 상태조회 연동 완료

- 공공데이터포털 **데이터셋 1순위** (`nts-businessman/v1/status`) 연동
- 사업자번호 10자리 입력 → 국세청 API 실시간 조회
- 조회 결과 DB upsert (`ON CONFLICT (biz_no) DO UPDATE`)
- **24시간 캐시**: DB에 저장된 데이터가 24h 이내면 국세청 재호출 없이 반환 (`cached: true`)
- 응답 필드: `b_no`, `b_stt`, `b_stt_cd`, `tax_type`, `end_dt`, `cached`
